## Description
This repo was created to learn VueJS! 

## Old build procedure
``` bash
# Clone the repo
git clone https://github.com/daxAKAhackerman/test-project

# Install dependencies
npm install

# Fix any vulns
npm audit fix

# Lint the code
npm run lint

# Build the app
npm run build
```
